## Modding Policy

This project **actively supports modding**.

Mods are permitted only under the following conditions:

### Permitted Mods

- Mods that operate **on top of the Official Build**
- Plugins, scripts, or data/resource replacements
- Patch-style modifications applied to the official release
- Mods loaded through official or community-supported modding mechanisms

### Not Considered Mods

- Any distribution containing the main executable or core binaries
- Mods that include substantial portions of this repository’s source code
- Standalone builds that function without the Official Build
- Distributions that effectively replace or replicate the base game

Original content created by mod authors remains the property of its creators, provided that it does not infringe on third-party rights.

For technical details, see `MODDING.md`.
